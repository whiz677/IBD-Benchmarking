# Dataset Notes

## Primary CELLxGENE collection
Human inflammatory bowel disease (IBD) and healthy control 10x Single-cell transcriptomics data.

CELLxGENE collection ID:
7c7bd6c2-925b-4034-baab-620ef1b760e1

Publication:
Garrido-Trigo et al., 2023, Nature Communications.

GEO:
GSE214695

Tissue:
Colonic mucosa.

Disease groups:
Healthy control, ulcerative colitis, Crohn's disease.

Starting task:
IBD vs healthy control.

Secondary task:
UC vs CD.

Why this dataset:
It is gut-specific, disease-relevant, interpretable, and small enough for a home PC.
