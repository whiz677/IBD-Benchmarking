# IBD Biology Notes for Validation

## Disease biology to test
1. Myeloid inflammation
   - macrophages
   - monocytes
   - neutrophils

2. Epithelial stress and barrier dysfunction
   - epithelial cells
   - goblet cells
   - absorptive cells

3. Stromal remodeling
   - fibroblasts
   - inflammatory fibroblasts

4. Immune activation
   - T cells
   - B cells
   - plasma cells

## Gene sets used in this starter project
TNF/NF-kB / inflammatory:
TNF, NFKBIA, CXCL8, IL1B, CXCL2, CXCL3

Interferon:
ISG15, IFIT1, IFIT3, MX1, OAS1, IFI6

Antigen presentation:
HLA-DRA, HLA-DRB1, HLA-DPA1, HLA-DPB1, CD74

Epithelial stress:
REG1A, REG1B, DUOX2, LCN2, MUC1

## Interpretation
If model-predicted IBD scores correlate with these pathway scores inside relevant cell types, the embedding may reflect real disease biology. If scores correlate more with donor/dataset/assay, the result may be confounded.
